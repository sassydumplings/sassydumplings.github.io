Rails Projects

I Need a Drink

The I need a Drink site was designed to find the right beverage for time of day. Drink establishments are displayed using the Google Maps API. I used Yelp APIs to search for local establishments. In the morning you will see a list of coffee places. Cafes are listed at lunchtime. The site will list the 10 nearest bars after 3 pm.
Note: this website is hosted on free Heroku services so will load very slowly.

Technology: Javascript,HTML,CSS,Ruby,Yelp API,GoogleMaps API

Kohwop

Kohwop is a social media site for Homeschooling families. In our due diligence, we learned that many homeschoolers felt isolated. We built a site to make it easy to share resoures, set up field trips and study dates. We also included lesson planning and skills tracking features. We surveyed several local homeschooling groups to validate our feature list.

Technology: Ruby on Rails, Javascript, HAML, Twitter Bootstrap, RubyGems:Devise,Forem,GeoCoder,Leaflet
